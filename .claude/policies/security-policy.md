# Security Policy — bkit-doctor

## Hard Rules
| 항목 | 규칙 |
|------|------|
| 자동 배포 | 절대 금지 |
| 코드 외부 전송 | 금지 (어떤 형태로도) |
| 외부 서비스 자동 연동 | 기본값 금지, 명시적 요청 시만 |
| 시크릿/토큰 하드코딩 | 금지 |
| `git push --force` | 명시적 요청 없으면 금지 |
| `rm -rf` | 명시적 요청 + 경로 확인 후만 허용 |

## Hook Restrictions
- 자동 배포 트리거 훅 금지
- 외부 네트워크 전송 훅 금지
- 코드 파일 외부 복사 훅 금지

## Data Classification
- `.claude/` 내 파일: 프로젝트 내부 전용
- `src/` 코드: 외부 공유 전 명시적 승인 필요
- API 키, 토큰, 비밀번호: 절대 코드에 포함 금지
